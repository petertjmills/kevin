# kevin
Prescription calendar, reminds users when they need to take their medication

# Dev setup

## MacOS:

### Install expo

```
npm install -g expo-cli
```

(?) and 


```
yarn add expo
```

### Clone Repo

```
git clone https://github.com/petertjmills/kevin.git
cd kevin
```

### Start expo server
```
expo start
```
