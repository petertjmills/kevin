# Dev setup

## MacOS:

### Install expo

```
npm install -g expo-cli
```

(?) and 


```
yarn add expo
```

### Clone Repo

```
git clone https://github.com/petertjmills/kevin.git
cd kevin
```

### Start expo server
```
expo start
```

### On the Expo Go app on Android
Scan the QR code in console, or on the web GUI

The app should load!
